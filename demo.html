Default setup
$('#t1').autoheight();
Custom params
$('#t2').autoheight({maxHeight: 150, minHeight: 50});
Make it beautiful with CSS3 transition
$('#t3').autoheight();
#t3 {
 transition-property: height;
 transition-duration: 100ms;
}
Use a callback
$('#t4').autoheight(function() {
  $('#spCallback').show().fadeOut(700);
}); 		callback called!
Use a callback
$('#t5').autoheight({maxHeight: 150}, function(h) {
  $('#txtCallback').val($(this).val()).height(h);
});
